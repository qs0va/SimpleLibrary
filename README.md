# SimpleLibrary
Test task for Dynamika

Настройка перед запуском:

Все что необходимо - указать реквизиты для подключения к БД.

Можно сделать в файле application.properties, который расположить в одной директории с .jar, в виде:

spring.datasource.url=*jdbc url*
spring.datasource.username=*db user*
spring.datasource.password=*password*

По умолчанию (внутри пакета) указаны следующие значения:

spring.datasource.url=jdbc:postgresql://localhost:5432/db_simplelibrary
spring.datasource.username=user_simplelibrary
spring.datasource.password=postgres

Сервер запускается на порте 8080 по умолчанию, в том же application.properties можно указать server.port=*custom port*.

Список книг доступен по пути /books, список клиентов - по пути /clients, дальнейшая навигация с помощью ссылок и кнопок.

Чтобы взять книгу на прочтение - нажать соответствующую кнопку в списке книг, затем выбрать читателя из клиентов. Поддерживается также обратный порядок - сначала выбрать читателя в обзем списке, а затем книгу.

json со списком читающих клиентов доступен по адресу /borrow/all (только через адресную строку, в веб-интерфейсе адрес не используется)

Валидация, ограничения на ввод и обработка ошибок не реализованы (в задании не было), в любое поле можно ввести любое значение (в тч пустое), при ошибке выпадет стандартная страница.
